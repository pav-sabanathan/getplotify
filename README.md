# 📺 Episode Tracker & Release Calendar

A personal TV episode tracker that brings together release schedules 
across all major streaming platforms into a single calendar view.

**Status:** In Planning  
**Built with:** Lovable  
**PM Documentation:** See PRD.md for full product thinking

---

## The Problem
Streaming is fragmented. Shows release across Netflix, Disney+, 
Apple TV+, Prime Video, BBC iPlayer and more — with no single place 
to track when new episodes drop. This tool solves that.
