# Research

This folder contains user research conducted throughout the 
development of Plotify — including discovery notes, user 
interview findings, and insights that informed product decisions.

---

## What You'll Find Here

As Plotify develops, this folder will contain:

- **Discovery notes** — early problem validation and assumption mapping
- **User interview guides** — discussion guides used in research sessions
- **Research findings** — synthesised insights from user conversations
- **Survey results** — any quantitative data collected from users
- **Usability notes** — observations from testing the product with 
  real users

---

## Current Status

Plotify V1 was built using the PM as the primary user — 
the problem was validated personally before any external 
research was conducted. Formal user research with external 
participants is planned as part of the Pre-V2 process to 
validate assumptions before the next build sprint.
